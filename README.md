GUI for Lakeshore 330/336/340 and Oxford ITC503, live monitoring of current temperature, and set the setpoint temperature + control of Temperature ramp, heater range.

You are free to extend the gui with other controls, here is the basic stuff I needed in my lab.

It should work with Lakeshore 330, 336, 340 (main features only). My gui is tested with Lakeshore 340 and Oxford ITC503, connected via GPIB. PyVisa is used for connection.

Have fun ;))

Class is based on the file by Richard Ziegahn: https://github.com/RickyZiegahn/Lakeshore-Cryostat-Controller
